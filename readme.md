## Installation
This app was wrapped using Native Code of React Native. Install the React Native CLI, follow the official Getting Started [here](https://facebook.github.io/react-native/docs/getting-started.html). Clone the repo, open it with any [code](blog.bkdev.me/2017/04/11/vscode.html) editors and you are ready to have fun üçü.

```
$ yarn
$ yarn start
$ react-native run-ios
$ react-native run-android
```
